Java处理字符串-删除所有标点

代码

在这里利用的是Unicode编码，Unicode 编码并不只是为某个字符简单定义了一个编码，而且还将其进行了归类。

\pP 其中的小写 p 是 property 的意思，表示 Unicode 属性，用于 Unicode 正表达式的前缀。

大写 P 表示 Unicode 字符集七个字符属性之一：标点字符。其他六个包括：

  • L：字母
  • M：标记符号（一般不会单独出现）
  • Z：分隔符（比如空格、换行等）
  • S：符号（比如数学符号、货币符号等）
  • N：数字（比如阿拉伯数字、罗马数字等）
  • C：其他字符

Java 中用于 Unicode 的正则表达式数据都是由 Unicode 组织提供的。
